# VitalRank

Сервис SEO-аудита сайта. Принимает URL, собирает технические данные, скорость
и Core Web Vitals, и выдаёт рекомендации двумя треками: по логике Google и по
логике Яндекса, отсортированные по соотношению эффект/усилие.

## Стек

- API-сервер: FastAPI (Python)
- Очередь задач: Celery + Redis
- База данных: PostgreSQL
- Аудит-движок (воркер): сбор данных и два анализатора

## Структура

```
backend/    API-сервер: маршруты, модели обращений к БД, авторизация
shared/     общий код: модели SQLAlchemy, подключение к БД
postgres/   инициализация БД
redis/      конфиг брокера
nginx/      прокси для фронта
```

## Запуск

Нужен установленный Docker.

1. Скопировать `backend/.env.example` в `backend/.env` и заполнить значения
   (секрет JWT, ключ PageSpeed).
2. Поднять стек:

```
docker compose up -d
```

3. Открыть документацию API: http://localhost:8000/docs

## Проверка

В Swagger (`/docs`) можно зарегистрироваться (`POST /auth/register`), войти
(`POST /auth/login`), нажать Authorize и дальше дёргать защищённые маршруты:
запуск аудита, список сайтов, история, отчёт, экспорт в PDF.

Готовый пример отчёта без авторизации: `GET /demo/report`.

## Тесты
